## 2. 测试场景描述

### 2.1 测试目标

本次性能压测旨在验证 UAV Platform V2 在以下场景下的性能表现：

- 验证各核心 API 在不同并发级别下的响应能力和稳定性
- 识别系统性能瓶颈和扩展性限制
- 验证 SLA 目标（P99 < 500ms, 可用性 >= 99.9%）
- 评估系统在高负载下的资源消耗情况

### 2.2 测试端点

| 端点名称 | 方法 | 路径 | 描述 | 类别 |
|----------|------|------|------|------|
| algorithm-list | GET | /api/v1/algorithm/list | 算法列表查询 | 查询 |
| health-check | GET | /actuator/health | 健康检查 | 基础设施 |
| weather-point | POST | /api/v1/weather/point | 气象点查询 | 数据服务 |
| planning-path | POST | /api/v1/planning/path | 路径规划 | 核心算法 |
| assimilation-task | POST | /api/v1/assimilation/task | 同化任务 | 核心算法 |
| risk-assessment | POST | /api/v1/risk/assess | 风险评估 | 数据服务 |

### 2.3 测试策略
